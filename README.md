## 🚧 What I'm building

**[🧠 Jamna · Java Neural Nets](https://github.com/vishnu-vardhan-d/jamna.nn)**
A mini Transformer / LLM engine written from scratch in **pure Java — no libraries**.
A hand-written autograd engine, multi-head attention, LayerNorm, and Adam; it learns to sort
by next-token prediction. Ships with an interactive explainer on GitHub Pages.

**[📈 Options Wall Street](https://github.com/vishnu-vardhan-d/options-wall-street)**
A full app I'm building to **automate options trading** for daily use — strategy automation,
payoff & P&L tracking, and execution. **Spring Boot** back end + **Next.js / React** front end,
working with live market data.
